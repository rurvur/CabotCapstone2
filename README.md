# Title

Cornursery

## Abstract

In the indie gaming sphere, there has been a steady rise of controlled distraction games in recent years. These games, often indicated by a very slow rate of progression and a constant presence over other windows, seek to address fatigue by advertising themselves as a side activity while users do other things. In this project I aimed to create one of these games, while attempting to address some of the design flaws I believe the mainstream options struggle with.
